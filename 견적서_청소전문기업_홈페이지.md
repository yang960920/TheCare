# 청소 전문 기업 홈페이지 시안 견적서

---

## 1. 사용자 홈페이지 (프론트엔드) — **300,000원**

> 방문 고객이 직접 보는 기업 홈페이지입니다.
> 반응형(모바일/태블릿/데스크탑) 대응, 모던 UI/UX 설계 포함.

| 기능 | 설명 |
|------|------|
| **히어로 섹션** | 대형 배경 이미지 + 헤드라인 + CTA 버튼 2개 (견적 문의 유도) |
| **서비스 소개** | 줄눈 시공 / 입주 청소 / 탄성 코트 / 나노 코팅 / 새집증후군 제거 — 아이콘 카드 5종 |
| **회사 소개** | 기업 소개 텍스트, 핵심 통계 (누적 시공 건수·고객 만족도·업계 경력) |
| **고객 후기** | 서비스별 필터링 탭, 별점 표시, 후기 상세 클릭 시 **관리자 답변** 확인 가능 |
| **견적 문의** | 이름 / 연락처 / 서비스 선택 / 면적 / 메모 입력 폼 + 전화·카톡·방문 상담 안내 |
| **아카데미** | 기초·전문·마스터 3단계 수강 과정 카드 + 수강 상담 CTA |
| **포인트 적립** | 적립 방법 / 사용 방법 안내 + 보유 포인트 시각화 + 이벤트 배너 |
| **GNB / 푸터** | 스크롤 시 배경 전환 헤더, 모바일 햄버거 메뉴, 4컬럼 푸터 (회사 정보·링크·연락처) |
| **스크롤 애니메이션** | 전 섹션 Framer Motion 기반 fade-in + slide-up 진입 효과 |
| **반응형 디자인** | 모바일 → 태블릿 → 데스크탑 3단계 반응형 레이아웃 |

## 2. 관리자 CMS (어드민 패널) — **500,000원**

> 홈페이지 콘텐츠를 직접 수정·관리할 수 있는 관리자 전용 페이지입니다.
> 별도 레이아웃 (사이드바 + 상단 헤더), 모바일 드로어 메뉴 대응.

| 기능 | 설명 |
|------|------|
| **대시보드** | 요약 카드 4종 (견적 문의 수 / 후기 수 / 진행 과정 수 / 활성 이벤트) + 월별 문의 추이 차트 + 서비스별 비율 차트 + 최근 문의 목록 |
| **히어로 관리** | 메인 헤드라인·서브 카피·CTA 버튼 텍스트/링크·배경 이미지 URL 편집 + **실시간 미리보기** |
| **서비스 관리** | 서비스 5종 목록 테이블, 각 항목 이름·설명·이미지·노출 여부 수정, 순서 변경 핸들 |
| **후기 관리** | 후기 목록 (서비스별·별점별 필터), 신규 등록 모달, 노출 토글, 삭제, **관리자 답변 작성** (사용자 페이지에 자동 반영) |
| **아카데미 관리** | 수강 과정 카드 목록, 신규 등록·수정·삭제, 과정명·기간·수강료·정원·설명·노출 관리 |
| **포인트·이벤트** | 이벤트 배너 등록·수정·삭제 (제목·설명·기간·배경 컬러·노출), 포인트 적립 정책 텍스트 편집 |
| **견적 문의 수신함** | 접수 목록 테이블, 처리 상태 변경 (미확인→확인→완료), 행 클릭 시 상세 모달, **연락 드린 날짜 및 상담 메모** 기록, CSV 내보내기 |
| **회사 정보** | 회사명·대표·사업자번호·주소·연락처·이메일 편집, 홈페이지 노출 통계 수치 편집 |
| **팝업·공지** | 팝업 등록·수정·삭제, 이미지·링크·노출 기간·노출 위치 선택, 오늘 하루 안 보기 옵션 |
| **공통 기능** | 저장/삭제 시 토스트 알림 피드백, 삭제 확인 모달, 노출 토글 스위치, 모바일 반응형 |

## 기술 스택

Next.js 14 (App Router) · TypeScript · Tailwind CSS · Framer Motion · Zustand · React Hook Form · Recharts · Vercel 배포

## 비고

- 실제 이미지 교체 정도의 수정만 필요한 **시안 수준** 완성도
- DB 연동·결제·인증 등 실제 기능 로직은 별도 견적
- Vercel 무료 배포 기준, 추가 인프라 비용 없음
